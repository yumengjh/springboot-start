运行时 CORS 策略

修改后立即生效。多个域名或方法用英文逗号分隔；本地测试可使用 *，生产应填写明确域名。

运行时接口策略

禁用和接口级限流都使用 Ant 路径模式；多个模式以英文逗号分隔。运行时配置接口本身不会被禁用。

运行时 IP 访问策略

白名单非空时，仅其中地址或 CIDR 可访问；黑名单优先级更高。示例：127.0.0.1,192.168.1.0/24。切勿在远程环境误封自己的管理 IP。

持久化 IP 访问规则

此处是数据库规则；DENY 优先。测试时不要添加当前客户端的有效 DENY 或不含自身 IP 的 ALLOW。
